# SHC Classic V1 Research Notes

## English
This repository collects research notes and technical analysis for the SHC Classic V1 device ecosystem.
It is a living record of reverse engineering steps, findings, and open questions.
The focus is on technical documentation and evidence-based analysis.

Scope:
- Hardware and firmware structure (ROM/XIP, NAND layout, registry hives)
- Crypto material and certificate stores (KeyVault, cert chains, PFX usage)
- Update pipeline (NK image + USB ZIP flow)
- Internal APIs and services (native and managed components)
- RF protocol hints and device identity handling

Status:
- Documentation is active and evolving as new artifacts are extracted.
- Some findings are inferred from decompiled code and require device-side validation.
